Cara Hitung Posisi per Meja:
1. Koben input jumlah kemenangan tiap pemain dalam sesi (total semua game)
2. Pemain dengan menang terbanyak = Raja Meja 👑 (+3)
3. Pemain menang ≥ 2 game (bukan terbanyak) = Nyaris Sempurna 😤 (+2)
4. Pemain menang 1 game = Cari Jodoh 😏 (+1) · Menang 0 game = Penonton Setia ⬜ (+1)
5. Jika seri (2 pemain menang sama banyak) → keduanya dapat poin posisi tersebut
Bonus: Streak Raja Meja 3 sesi berturut = +2 · Hadir 4 sesi/bulan = +3

💡 Tips Pemula
✓Fokus 1 strategi — semua Chi atau semua Pong
✓Simpan Naga & Angin jika sudah punya 2 (incaran Pong)
✓Buang tile terisolasi (tidak punya pasangan) lebih awal
✓Perhatikan tile buangan lawan untuk tebak tangan mereka
✗Jangan ubah strategi di tengah permainan
✗Jangan buang tile yang sedang dicari lawan
✗Jangan tunggu tile "sempurna" jika tangan sudah hampir selesai
